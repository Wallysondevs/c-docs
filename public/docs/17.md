# C17

**ISO/IEC 9899:2018**, também conhecido como **C17**/**C18** (denotam o ano de conclusão e publicação, respectivamente), é uma revisão anterior do padrão C.

O C17 é o mesmo que o C11, exceto que ele eleva a macro predefinida [`__STDC_VERSION__`](<#/doc/preprocessor/replace>) para 201710L, contém vários relatórios de defeitos e deprecia algumas funcionalidades.

## Funcionalidades obsoletas

### Funcionalidades depreciadas

  * [ATOMIC_VAR_INIT](<#/doc/atomic/ATOMIC_VAR_INIT>)
  * Suporte para chamar [`realloc()`](<#/doc/memory/realloc>) com tamanho zero
