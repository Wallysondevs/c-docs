# Palavras-chave C: fortran

### Uso

  * especificador de tipo suportado condicionalmente para ligação de linguagem Fortran. Pode ser usado com [declarações de função](<#/doc/language/function_declaration>) e outras [declarações externas](<#/doc/language/extern>) para indicar que a convenção de chamada e o *name mangling* são adequados para ligação com unidades de tradução escritas na linguagem de programação Fortran.
