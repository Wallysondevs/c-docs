# Recursos úteis

### Perguntas Frequentes sobre C
  
  * [C FAQ](<https://c-faq.com/>) — Respostas para as perguntas frequentes do grupo de notícias Usenet **comp.lang.c**.

### Referências da Linguagem C e da Biblioteca

  * [JTC1/SC22/WG14](<https://www.open-std.org/jtc1/sc22/wg14/>) — O Comitê de Padrões C, inclui rascunhos de padrões da linguagem C, propostas e relatórios de defeitos.
    * [c89](<https://port70.net/~nsz/c/c89/c89-draft.html>) — Rascunho C89 (html).
    * [n1124.pdf](<https://open-std.org/JTC1/SC22/WG14/www/docs/n1124.pdf>) — Rascunho C99 (com TC1 e TC2).
    * [n1256.pdf](<https://open-std.org/JTC1/SC22/WG14/www/docs/n1256.pdf>) — Rascunho C99 (com TC1, TC2 e TC3).
    * [n1570.pdf](<https://open-std.org/JTC1/SC22/WG14/www/docs/n1570.pdf>) — Rascunho de trabalho final do C11.
    * [n2176 (C17 FDIS)](<https://files.lhmouse.com/standards/ISO%20C%20N2176.pdf>) — Rascunho final C17/C18.
    * [n3220.pdf](<https://open-std.org/JTC1/SC22/WG14/www/docs/n3220.pdf>) — Primeiro rascunho pós-publicação do C23 (contém apenas [correções editoriais para C23](<https://www.open-std.org/jtc1/sc22/wg14/www/docs/n3221.htm>)).
    * [n3435.pdf](<https://open-std.org/JTC1/SC22/WG14/www/docs/n3435.pdf>) — Último rascunho de trabalho público do C (2025-01-03).
  * [comp.std.c](<https://groups.google.com/forum/m/?hl=en#!forum/comp.std.c>) — Discussão sobre os padrões da linguagem C.
  * [GNU C Manual](<https://www.gnu.org/software/gnu-c-manual/gnu-c-manual.html>) — Referência da linguagem C central conforme implementada pelo compilador GNU C.

### ABIs C

  * [System V ABI](<https://github.com/hjl-tools/x86-psABI/wiki/X86-psABI>) — Para arquiteturas x86-64, Intel386 e IA MCU.
  * [Windows x64 ABI](<https://docs.microsoft.com/en-us/cpp/build/x64-calling-convention>)
  * [PowerPC EABI](<https://www.nxp.com.cn/docs/en/application-note/PPCEABI.pdf>) — Para processadores embarcados de 32 bits.
  * [PowerOpen ABI](<ftp://www.sourceware.org/pub/binutils/ppc-docs/ppc-poweropen/>) — Para plataformas PowerPC mais antigas.
  * [ARM ABI](<https://github.com/ARM-software/abi-aa/releases>) — Para arquitetura ARM.

### Padrões relacionados

  * [POSIX.1-2024/IEEE Std 1003.1-2024](<https://pubs.opengroup.org/onlinepubs/9799919799/functions/contents.html>) — A API do sistema operacional de sistemas compatíveis com Unix.
  * [Páginas man do Linux](<https://man7.org/linux/man-pages/index.html>) — A documentação da API do sistema operacional Linux.
