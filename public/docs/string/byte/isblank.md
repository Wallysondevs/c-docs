# isblank

Definido no cabeçalho [`<ctype.h>`](<#/doc/string/byte>)

```c
int isblank( int ch );  // desde C99
```

Verifica se o caractere fornecido é um caractere em branco na localidade C atual. Na localidade C padrão, apenas o espaço (`0x20`) e a tabulação horizontal (`0x09`) são classificados como em branco.

O comportamento é indefinido se o valor de `ch` não for representável como unsigned char e não for igual a [EOF](<#/doc/io>).

### Parâmetros

- **ch** — caractere a ser classificado

### Valor de retorno

Valor diferente de zero se o caractere for um caractere em branco, zero caso contrário.

### Exemplo

Execute este código
```c
    #include <stdio.h>
    #include <ctype.h>
    #include <limits.h>
    
    int main(void)
    {
        for (int ndx=0; ndx<=UCHAR_MAX; ndx++)
            if (isblank(ndx)) printf("0x%02x\n", ndx);
    }
```

Saída:
```
    0x09
    0x20
```
