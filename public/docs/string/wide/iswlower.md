# iswlower

Definido no cabeçalho [`<wctype.h>`](<#/doc/string/wide>)

```c
int iswlower( wint_t ch );  // desde C95
```

Verifica se o caractere largo fornecido é uma letra minúscula, ou seja, uma das `abcdefghijklmnopqrstuvwxyz` ou qualquer letra minúscula específica do locale atual.

### Parâmetros

- **ch** — caractere largo

### Valor de retorno

Valor diferente de zero se o caractere largo for uma letra minúscula, zero caso contrário.

### Observações

[ISO 30112](<https://www.open-std.org/JTC1/SC35/WG5/docs/30112d10.pdf>) especifica quais caracteres Unicode estão incluídos na categoria POSIX lower.

### Exemplo

Execute este código
```c
    #include <locale.h>
    #include <stdio.h>
    #include <wchar.h>
    #include <wctype.h>
     
    int main(void)
    {
        wchar_t c = L'\u0444'; // Cyrillic small letter ef ('ф')
        printf("In the default locale, iswlower(%#x) = %d\n", c, !!iswlower(c));
        setlocale(LC_ALL, "en_US.utf8");
        printf("In Unicode locale, iswlower(%#x) = %d\n", c, !!iswlower(c));
    }
```

Saída:
```
    In the default locale, iswlower(0x444) = 0
    In Unicode locale, iswlower(0x444) = 1
```
